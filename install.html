# Installation

Shell Scripts provides Bash and PowerShell install scripts to download any collection of scripts from the repository. The following command shows the list of downloadable scripts from the repository.

The following command will install the packup script. Other scripts can be installed by replacing the packup argument.

On Windows, PowerShell will need to run as administrator if the --user flag is not used and the security policy must allow for running remote PowerShell scripts. If needed, the following command will update the security policy for the current user.

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
